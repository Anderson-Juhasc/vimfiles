Instalação:

    git clone git://github.com/Anderson-Juhasc/vimfiles.git ~/.vim


Entrar no diretório `~/.vim`, e iniciar os submódulos:

    cd ~/.vim
    git submodule init
    git submodule update

Voltar para `~/` e criar um arquivo `.vimrc` e adicionar o seguinte texto:

	 source ~/.vim/vimrc
